# clove
clove is an [imdb](https://www.imdb.com) and [vidsrc](https://vidsrc.to) scraper written in clojure


## Installation
Download from http://github.com/71zenith/clove/releases/


## Usage
```sh
$ java -jar clove.jar [args]
```

## Examples

```sh
$ java -jar clove.jar "Kung Fu Panda 1"
```
